Here are brief descriptions of each program:

The Practice Programming Test was taken and completed according to test requirements in under two hours and expands upon the existing Stack data type with the implementation of two more methods.

Programming Assignment 1 is a simulation of a Computer Science department's professor management system, and implements Stack for the Faculty class to do so.

Programming Assignment 2 is a CD sorting system using an implementation of NodeBasedStack.

Programming Assignment 3 is a simulation of a Computer Science class' waitlist, using an implemenation of the Queue data type.

Programming Assignment 4 is an incomplete simulation of a Printer Prototype, with Main being incomplete and lacking error handling by extension.

Programming Assignment 5 is a simulation of a course management system, using an implementation of NodeBasedList to do so.

Every assignment includes a console UI and error handling, unless specified otherwise.
